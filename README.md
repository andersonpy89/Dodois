# Dodois

Aplicação para Controle Fincanceiro Familiar

# 🦤 Dodô

Dodô é uma aplicação web de **controle financeiro familiar** criada para registrar gastos, acompanhar categorias e entender melhor o fluxo de dinheiro ao longo do tempo.

A ideia do projeto nasceu de um problema cotidiano:  
acompanhar entradas, saídas e parcelamentos de forma simples e visual.

O objetivo foi construir uma ferramenta leve, direta e fácil de usar.

---

# 📸 Interface

Dashboard com visualização de:

- gastos por categoria
- forma de pagamento
- evolução do saldo
- entradas vs saídas

---

# 🚀 Funcionalidades

### 💸 Controle financeiro
- registro de **entradas**
- registro de **saídas**

### 🔁 Transações recorrentes
Permite criar gastos automáticos que se repetem ao longo dos meses.

Exemplo:

- aluguel
- assinaturas
- contas fixas

### 🧾 Parcelamentos
Distribui automaticamente valores em múltiplos meses.

Exemplo:


R$ 1200 em 6 parcelas
→ gera automaticamente 6 registros


### 📊 Dashboard financeiro

Gráficos com:

- gastos por categoria
- gastos por forma de pagamento
- evolução do saldo
- comparação entre entradas e saídas

### 📅 Resumo financeiro

- resumo mensal
- resumo anual
- melhor mês do ano
- pior mês do ano

---

# 🧠 Insights automáticos

O sistema identifica automaticamente:

- categoria com maior gasto
- forma de pagamento mais usada
- quem mais gastou (uso familiar)
